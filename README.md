# EGGS API 🥚
### EgalisysTech Great Geographical Service Platform – Backend

This repository contains the backend API for the **Egalisys Tech** service platform.

EGGS is a backend platform designed to support the operations of Egalisys Tech, a technology repair and support company. The platform will provide tools for users to request services and for managers to track and manage repair jobs.

---

## 🛠 Tech Stack

### Backend Framework
- NestJS

### Language
- TypeScript

### Database
- PostgreSQL

### Version Control
- Git / GitHub
